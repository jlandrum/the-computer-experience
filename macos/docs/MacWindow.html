focus() → {boolean}

Focuses the window. This will adjust the z-index of the window as well as update all windows to ensure the active window is on top without overflowing the z-indexes. If the window has the attribute `no-focus`, this will return false. If the window is already focused, or if it becomes focused this will return true.
